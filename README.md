# Wavelet_LSTM
本项目将**深度学习**与**数字信号处理**算法相结合，通过**LSTMs（RNN）**与**连续小波变换CWT**的松耦合提出CwtNet(连续小波长度时记忆网络)，实现了对结构健康体系的无损检测分析。欢迎大家提出宝贵的意见。
